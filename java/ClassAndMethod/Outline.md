スコープ：変数の有効範囲  
変数宣言されたブロック内がその変数のスコープ  
ブロック内の処理が完了すると、そのブロックをスコープとする変数はメモリ上から消去される  
スコープ外からアクセスしてもその変数は存在しないものとして扱われる  
ローカル変数：メソッド内で定義される変数  
・同スコープ内で同名の変数は定義できない  
  別スコープであれば可能  
・仮引数はその直後のブロック内に含まれる（メソッドブロック内がスコープ）
